# Lesson 15

## 01. Решение задачи со строками

* Реализовать словарь с подсказками. Пользователь вводит слово, если для него есть подсказки - то они предлагаются, если нет - слово добавляется в словарь

```
мед
  -> медь
  -> медведка
  -> мед
  -> медаль
```

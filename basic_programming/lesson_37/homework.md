## Task 37*

```
1. Создать классы Fruit, Apple extends Fruit, Orange extends Fruit;
2. Создать класс Box, в который можно складывать фрукты. Коробки должны соответствовать типу фрукта, поэтому в одну коробку нельзя сложить и яблоки, и апельсины;
3. Для хранения фруктов внутри коробки можно использовать ArrayList;
4. Сделать метод getWeight(), который высчитывает вес коробки, зная вес одного фрукта и их количество: вес яблока – от 1.0 до 1.2f, апельсина – от 1.0 до 1.5f (единицы измерения не важны);
5. Внутри класса Box сделать метод compareTo(), который позволяет сравнить текущую коробку с той, которую подадут в compareTo() в качестве параметра. -1 – если первая коробка весит меньше второй, 0 если они равны по весу, 1 если первая коробка весит больше. Можно сравнивать коробки с яблоками и апельсинами;
6. Написать метод moveTo(), который позволяет пересыпать фрукты из текущей коробки в другую. Помним про сортировку фруктов: нельзя яблоки высыпать в коробку с апельсинами. Соответственно, в текущей коробке фруктов не остается, а в другую перекидываются объекты, которые были в первой;
7. Не забываем про метод добавления фрукта add() в коробку.
```
